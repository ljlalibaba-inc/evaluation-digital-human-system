# 人设特征说明

## 各人设典型特征

### P001 - 新手焦虑型妈妈 ⭐ (特征最明显)

**核心特征：**
- 😰 **情境描述**：85% 概率出现（孩子发烧、宝宝不舒服等）
- 💭 **情感表达**：70% 概率出现（着急、担心、焦虑等）
- 🙏 **礼貌用语**：30% 概率（辛苦推荐、求推荐等）

**典型 Query：**
```
✅ 孩子发烧，推荐飞鹤的奶粉
✅ 宝宝不舒服，找辅食
✅ 孩子咳嗽，找 200 元以内分体童装
✅ 小孩拉肚子，需要童装 10 公里内，要求急送
❌ 推荐奶粉（缺少情境和情感，不符合人设）
```

---

### P002 - 效率实用型妈妈

**核心特征：**
- ⚡ **简洁直接**：短句为主
- 📍 **地理位置**：80% 概率（X 公里内、附近等）
- 💰 **价格范围**：50% 概率
- ❌ **无情境/情感**：直奔主题

**典型 Query：**
```
✅ 性价比高的奶粉
✅ 200 元以内纸尿裤 3 公里内
✅ 今天纸尿裤 5 公里内
✅ 玩具 10 公里内
```

---

### P004 - 价格敏感型妈妈

**核心特征：**
- 💰 **价格范围**：80% 概率出现
- 💵 **价格词汇**：便宜点、实惠的、有优惠吗
- ❌ **无情境/情感**：关注价格本身

**典型 Query：**
```
✅ 想买便宜点的童装
✅ 婴儿车有优惠吗
✅ 100 元以内纸尿裤推荐一下
✅ 200 元以内的玩具哪里买
```

---

### P003 - 品质追求型妈妈

**核心特征：**
- 💎 **品质导向**：高品质的、专业的
- 🎯 **品牌偏好**：90% 概率
- 💭 **情感表达**：注重品质、追求体验

**典型 Query：**
```
✅ 求推荐高品质的贝因美的奶粉
✅ 婴儿车哪个昆塔斯的好
✅ 想买专业的早教玩具
```

## 人设识别要点

### 新手焦虑型妈妈
看到以下特征即可识别：
1. ✅ 开头有情境界定（孩子发烧、宝宝不舒服）
2. ✅ 中间有情感表达（着急、担心、焦虑）
3. ✅ 结尾有礼貌用语（辛苦推荐、求推荐）

**示例：**
```
"孩子发烧，着急，推荐美赞臣的 1 段奶粉 3 公里内，辛苦推荐"
 ↑情境    ↑情感         ↑品牌规格       ↑位置   ↑礼貌
```

### 效率实用型妈妈
1. ✅ 简短直接
2. ✅ 明确的地理位置
3. ✅ 可能有价格范围

**示例：**
```
"200 元以内纸尿裤 3 公里内"
 ↑价格          ↑位置
```

### 价格敏感型妈妈
1. ✅ 必有价格相关词汇
2. ✅ 询问优惠/便宜

**示例：**
```
"100 元以内纸尿裤哪里买"
 ↑价格范围
```

## 测试验证

运行以下命令验证人设特征：

```bash
# Java
cd java && java QueryGenHttpServer 5000 &
curl -X POST http://localhost:5000/generate \
  -H "Content-Type: application/json" \
  -d '{"persona":"新手焦虑型妈妈","scene":"母婴用品","count":10}'

# Python
cd python && python3 query_gen_cli.py -p "新手焦虑型妈妈" -s "母婴用品" -c 10
```

- 新手焦虑型妈妈：8-9/10 条包含情境描述
- 效率实用型妈妈：简洁 + 地理位置
- 价格敏感型妈妈：4-5/10 条包含价格范围
